# 字段设计：文章审核与发布前准备

## 核心用途

字段用于记录每篇公众号文章从选题、AI 生成、人工审核到发布前准备的状态。

## 推荐字段

| 字段名 | 字段类型 | 说明 |
|---|---|---|
| 任务编号 | 文本 | 每篇文章唯一编号 |
| 创建日期 | 日期 | 创建文章任务的日期 |
| 当前 IP 署名 | 文本 | 用于公众号署名或内部识别 |
| 文章主题 | 文本 | 本期文章主题 |
| 目标读者 | 文本 | 帮助 AI 控制表达方式 |
| 内容来源 | 文本 | 直接生成 / 知识库 / 每周计划 |
| 脱敏确认 | 复选框 | 确认不含真实身份信息 |
| 风险标签 | 多选 | 如自伤、自杀、未成年人、药物建议等 |
| 风险等级 | 单选 | 低 / 中 / 高 |
| AI 生成状态 | 单选 | 待输入 / 已生成 / 重新生成中 |
| 审核状态 | 单选 | 待审核 / 审核通过 / 退回修改 / 不通过 |
| 审核意见 | 长文本 | 记录退回修改意见 |
| 固定声明检查 | 复选框 | 检查文章结尾两句话 |
| 排版状态 | 单选 | 未排版 / 排版中 / 已发布 |
| 发布时间 | 日期时间 | 人工发布完成时间 |
| 异常记录 | 长文本 | 记录问题 |

## 状态流转

1. 新建任务
2. AI 生成文章
3. 风险审核
4. 人工审核
5. 退回修改或审核通过
6. 复制 Markdown
7. 打开 Raphael 排版预览
8. 手动粘贴到公众号后台
9. 人工发布
10. 标记为已发布

## 拦截规则

- 未脱敏的内容不得进入 AI 生成。
- 未审核通过的文章不得进入发布前准备。
- 缺少固定声明的文章不得发布。
- 高风险主题必须人工重点审核。
- 系统不自动发布、发表或群发公众号文章。

## 一句话原则

任务字段不是为了增加流程负担，而是为了确保每篇文章都经过脱敏、风险审核、人工确认和安全发布前准备。
