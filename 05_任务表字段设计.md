# 任务表字段设计

## 1. 任务表用途

任务表用于记录每篇公众号文章从选题、AI 生成、医生审核到 TeleClaw 保存草稿的状态。

第一版可以直接用 Excel、飞书表格、腾讯文档或 Google Sheets 管理，不需要开发系统。

## 2. 推荐字段总表

| 字段名 | 字段类型 | 是否必填 | 示例 | 说明 |
|---|---|---:|---|---|
| 任务编号 | 文本 | 是 | 20260529-001 | 每篇文章唯一编号 |
| 创建日期 | 日期 | 是 | 2026-05-29 | 医生创建选题的日期 |
| 医生署名 | 文本 | 是 | 张医生 | 用于公众号署名或内部识别 |
| 公众号栏目 | 单选 | 否 | 情绪科普 | 例如情绪、关系、睡眠、亲子等 |
| 科普主题 | 文本 | 是 | 为什么越累越睡不着 | 本期文章主题 |
| 目标读者 | 文本 | 否 | 经常焦虑、睡眠浅的人 | 帮助 AI 控制表达方式 |
| 内容来源类型 | 单选 | 是 | 脱敏后的常见咨询主题 | 不允许填写原始咨询记录 |
| 脱敏确认 | 复选框 | 是 | 已确认 | 医生确认不含身份信息 |
| 高风险主题 | 多选 | 是 | 自伤、未成年人 | 如无则填“无” |
| 风险等级 | 单选 | 是 | 低 / 中 / 高 | 由星辰智能体初判，医生复核 |
| AI 文章生成状态 | 单选 | 是 | 未生成 / 已生成 / 需重写 | 记录星辰智能体处理状态 |
| 风险审核报告状态 | 单选 | 是 | 未生成 / 已生成 / 需复核 | 记录风险报告状态 |
| 医生审核状态 | 单选 | 是 | 未审核 / 通过 / 需修改 / 不通过 | 只有“通过”才可交给 TeleClaw |
| 医生审核意见 | 长文本 | 否 | 第三段表述需更温和 | 记录修改意见 |
| 固定声明检查 | 复选框 | 是 | 已包含 | 检查文章结尾两句话 |
| TeleClaw 任务包状态 | 单选 | 是 | 未生成 / 已生成 / 禁止执行 | 医生通过后才可生成可执行任务包 |
| TeleClaw 执行状态 | 单选 | 是 | 未执行 / 已保存草稿 / 异常停止 | 只记录草稿保存状态 |
| 草稿保存时间 | 日期时间 | 否 | 2026-05-29 16:30 | TeleClaw 保存完成时间 |
| 是否发布 | 单选 | 是 | 未发布 / 医生已手动发布 | TeleClaw 不得修改此项为发布 |
| 异常记录 | 长文本 | 否 | 后台无法确认保存按钮 | 记录问题 |
| 最终备注 | 长文本 | 否 | 下次选题可延展睡眠系列 | 复盘或后续计划 |

## 4. 状态流转规则

推荐按以下顺序流转：

1. 新建任务
2. 医生填写脱敏科普卡
3. 星辰智能体生成文章和风险审核报告
4. 医生审核
5. 医生审核通过
6. 生成 TeleClaw 任务包
7. TeleClaw 保存公众号草稿
8. 医生手动发布或继续修改

## 5. 关键拦截规则

### 脱敏未确认

如果 `脱敏确认` 不是“已确认”，不得交给星辰智能体生成文章。

### 医生未审核通过

如果 `医生审核状态` 不是“通过”，不得生成可执行的 TeleClaw 任务包。

### 固定声明未包含

如果 `固定声明检查` 不是“已包含”，不得进入 TeleClaw 执行。

### 高风险主题

如果 `高风险主题` 不是“无”，必须要求医生重点审核。

### TeleClaw 执行边界

`TeleClaw 执行状态` 只能记录：

- 未执行
- 已保存草稿
- 异常停止

不能记录“已发布”“已群发”，因为这些动作不允许由 TeleClaw 执行。

## 6. 推荐表格视图

### 今日待处理

筛选条件：

- 创建日期 = 今天
- 医生审核状态 != 通过，或 TeleClaw 执行状态 = 未执行

### 待医生审核

筛选条件：

- AI 文章生成状态 = 已生成
- 风险审核报告状态 = 已生成
- 医生审核状态 = 未审核

### 高风险重点审核

筛选条件：

- 高风险主题 != 无

### 可交给 TeleClaw

筛选条件：

- 医生审核状态 = 通过
- 固定声明检查 = 已包含
- TeleClaw 任务包状态 = 已生成
- TeleClaw 执行状态 = 未执行

### 已保存草稿

筛选条件：

- TeleClaw 执行状态 = 已保存草稿
- 是否发布 = 未发布

## 8. 一句话使用原则

任务表不是为了增加流程负担，而是为了确保每篇文章都经过脱敏、风险审核、医生确认和安全保存草稿。
